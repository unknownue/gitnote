# Numpy Note

```Python
import numpy as np
```

### dot product 
```Python
np.dot(a, b, out=None)
```
- For 1D a, b tensor, `numpy.dot` is dot(scalar) product
- For 2D a, b tensor, `numpy.dot` is matrix-matrix product
